## About

This is a collection of reusable components that can be used in your project. These components are built using the shadcn design system and ui components.

You can copy, paste and edit these components according to your needs.

The project requires some setup to work properly. You can find the setup instructions in the [Getting Started](/docs/getting-started/installation) section.
